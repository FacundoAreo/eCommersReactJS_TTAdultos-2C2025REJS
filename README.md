# React eCommerce App

Una aplicación de comercio electrónico moderna construida con React.

## Características

- 🛍️ Catálogo de productos
- 🛒 Carrito de compras
- 🚀 React Router para navegación
- 📱 Diseño responsive
- 🔐 Rutas protegidas
- 🎨 Interfaz moderna

## Tecnologías

- React 18
- React Router DOM
- Context API
- CSS3
- GitHub CLI

## Instalación

\`\`\`bash
npm install
npm start
\`\`\`
